### NETWORK
**Create a network**
```
podman network create <mynetwork>
```
---
**List all networks**
```
podman network ls
```
---
**Delete a network**
```
podman network rm <mynetwork       // Deletes network
podman network rm -f <mynetwork>   // Deletes all containers and network
```
---
**Delete a network**
```
podman network inspect <network-name>   // Inspect a network
```
